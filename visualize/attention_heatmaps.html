Visualize attention weights

Input a pair of sentences for one of the available tasks to see the model prediction and visualize the attention weights for each head.

Warning: You are browsing a static version of this website. Since there is no backend it is not possible to run any models.

About this visualizer

Upon submitting a pair of sentences, the model will output a prediction and show the confidence on each label. It will also output an interactive heatmap of attention weights.

Each cell of the heatmap encodes the attention from the token in the y axis to the token in the x axis. The buttons panel allows to select a single head of a single layer. Clicking on a layer label will show the average weights of all heads in that layer. Clicking on a head label will show the average weights of the heads in that position in every layer. Clicking on the circle on the lower left of the buttons panel will show the average of all heads in every layer.
