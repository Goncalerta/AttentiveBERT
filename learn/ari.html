Argumentative Relation Identification

Argumentative Relation Identification (ARI) is a subtask of Argument Mining (AM) and aims at classifying if a given source Argumentative Discourse Unit (ADU) supports, attacks, or is unrelated to a second ADU [Peldszus and Stede, 2015].

Task

ARI is a classification task that is a subtask of the Argument Mining task. Argument mining is research area with the goal of automatically extract and identify argumentative structures. The ARI subtask consists of determining, for two sentences with argumentative content (also known as Argumentative Discourse Units), whether the first attacks, supports, or is unrelated to the second. The first sentence is known as Source ADU and the second as Target ADU.

Since it is a classification task, the model uses the output corresponding to the [CLS] to make the prediction. The prediction is done with a Feed Forward neural network and a softmax layer, giving the confidence of each possible classification label.

Datasets

Argumentative Essays Corpus (english dataset)

The Argumentative Essays Corpus (AE) was introduced in Parsing Argumentation Structures in Persuasive Essays.

ArgMine Corpus (portuguese dataset)

The ArgMine corpus is a portuguese dataset for ARI that envisions to automatically extract the argumentative reasoning included in textual documents, such as opinion articles.
