Natural Language Inference

Natural Language Inference (NLI), also known as Recognizing Textual Entailment (RTE), is the task of determining the inference relation between a text and an hypothesis: entailment, contradiction, or neutral [MacCartney and Manning, 2008].

Task

NLI is a classification task that receives two sentences as input and tries to determine the relationship between those sentences. This relationship can be entailment, contradiction, or neutral.

Since it is a classification task, the model uses the output corresponding to the [CLS] to make the prediction. The prediction is done with a Feed Forward neural network and a softmax layer, giving the confidence of each possible classification label.

Datasets

This section describes two well known datasets for NLI: the SNLI corpus and the MNLI corpus.

SNLI Corpus

The Stanford Natural Language Inference (SNLI) corpus is a collection of 570k human-written English sentence pairs for the NLI task. This dataset was introduced in A large annotated corpus for learning natural language inference.

MNLI Corpus

The Multi-Genre Natural Language Inference (MultiNLI) corpus is a crowd-sourced collection of 433k sentence pairs for NLI. It is modeled on the SNLI corpus, but covers multiple genres of spoken and written text. This dataset was introduced in A Broad-Coverage Challenge Corpus for Sentence Understanding through Inference.
